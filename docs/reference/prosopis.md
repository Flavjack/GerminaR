# Germination under different osmotic potentials and temperatures.

Dataset containing information from germination experiment with Prosopis
juliflor under different osmotic potentials and temperatures.

- `rep` a numeric vector, repetitions or replications.

- `nacl` a numeric vector, NaCl osmotic potentials levels in Mpa to be
  evaluated.

- `temp` a numeric vector, temperature levels in centigrades to be
  evelauted.

- `seeds` a numeric vector, number of seed used for experimental unit.

- `D0-D10` numeric vectors, ten evaluations days performed in the
  germination experiment.

## Usage

``` r
prosopis
```

## Format

A data frame with 15 columns and 80 rows.
